# Trip Planning App

This is a simple Android app made with Java.  
The app lets the user add, edit, view, and delete trip tasks.

## Features
- Add a new task
- Edit a task
- Delete a task
- View task details
- Save tasks using SharedPreferences
- Show tasks in a RecyclerView

## Main Files
- MainActivity.java → shows all tasks
- AddEditActivity.java → add or edit a task
- DetailActivity.java → show task details
- Task.java → model class
- TaskAdapter.java → RecyclerView adapter
- SharedPrefManager.java → save & load tasks

## How to Run
1. Open project in Android Studio  
2. Let Gradle sync  
3. Run on emulator or device
